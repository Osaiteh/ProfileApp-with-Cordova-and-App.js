Trainings

SheCanWithICT

This program was organised in 2017 and it lasted for 6 months. The program was organised by Women Technology Empowerment Centre (WTEC) in collaboration with Intel West Africa, University of Lagos(Unilag), MainOne, BudgIT, Goethe Institut, Andela, WikiLovesWomen, AshaProject, WISE Nigeria, WorldPulse, Exclusive Brands Africa(EBA), Irede Foundation and Slingshot Technologies. Even though the main aim was to teach women to incoporate ICT in their businesses, career and network, it was not limited to that.

This was where I first learnt about the great possibilities of technology, where my passion for technology began and this was also where I met my first model, Mrs. Yetunde Johnson, the MD of Slingshottechnologies.

Code Lagos

I attended CodeLagos in 2018. I learnt the basics of frontend web development using HTML, CSS and JavaScript.

I designed a basic restaurant website using HTML, CSS and JavaScript.

Andela Learning Community (ALC)

The ALC 4.0 was launched on the 15th of may 2019. The Program was organised by Andela Learning Community(AlC) in partnership with Google and Pluralsight. It's an online learning program with occassional physical meet-ups. The Programme was devided into 3 phases. For each phase, we had one meet-up, one test requirement and one project at least. At the end of the day, I grew from a Novice to an Expert, according to Pluralsight standards.
